# Historico de desarrollo

Esta carpeta conserva el arbol documental anterior a la reorganizacion operativa de 2026-07-12. Incluye requisitos fuente, decisiones, fases, diarios de avance, validaciones y documentos de modulos originales.

No se elimina para preservar la trazabilidad academica y tecnica del proyecto. La documentacion vigente y ordenada por modulo se encuentra en el directorio padre `docs/`.

Algunos enlaces internos del historico mantienen las rutas originales y pueden apuntar a documentos que ahora estan fuera de su ubicacion inicial. Para operacion, ejecucion y despliegue se debe usar el [README principal](../../README.md) y el [indice vigente](../00-indice.md).
