# Desarrollo por fases

La reconstruccion de Kontora POS se hara de forma secuencial:

1. Fase 1: proyecto, entorno Docker, PostgreSQL local y documentacion base.
2. Fase 2: backend Spring Boot, Maven, configuracion base, Flyway, JPA, seguridad base y salud.
3. Fase 3: backend modular, compilando y probando antes de avanzar de modulo.

No se debe avanzar a una fase o modulo si la fase anterior no quedo verificada en la maquina local.
