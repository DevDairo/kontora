# Schema canonico

Esta carpeta conserva el script fisico de la base de datos entregado para Kontora POS.

Archivo principal:

- `kontora_pos_schema_v1_1.sql`

Aunque el encabezado interno del SQL indique `Version: 1.0`, el archivo se conserva sin ediciones manuales para proteger la fuente de verdad. Cualquier cambio estructural futuro debe entrar como migracion nueva y revisada.
